PollEndpoint

About

It's for you to create questionnaires, surveys and polls in no time.

It's created for ease of use so if you have anything to complain please contact us.

About the creator

Hi! I'm the site's creator, Licson Lee. I've been working on some other projects before but they're not so useful so I'm not talking them at here. If you want to know more about me, you can check out my personal website.

I created PollEndpoint originally for my homework assignment because other online survey service doesn't match my requirements. So I made my own site for making online questionnaires and I found it pretty useful for others so I release it.

Hope you like it.

Motivation

Before the creation of PollEndpoint, I have done some online surveys and found most of them not easy to use and hardcoded.

Therefore, I want to create a survey service that is easy to use and manange. PollEndpoint is the final product that I've made.

Support & Ideas

Currently we have a facebook page to collect ideas and user support. If you have any problems while using PollEndpoint, feel free to post at the page or email me (licson0729@gmail.com).

If you found problems with my code or want to suggest new features, you can submit an issue on GitHub or you can post it to our facebook page.

PollEndpoint is open-source

I think I forget to tell you something. PollEndpoint is open-source. If you're interested about how I wrote this, you can have a look at the repository on GitHub
